
Currently, UnityPlugin for macOS cannot be notarized due to the circumstances of Unity.

現在、macOS向けのUnityPluginはUnityの都合により公証できない状態です。


Reference (参考):

https://forum.unity.com/threads/unity-macos-plugin-bundle-notarization-problem.866479/

When playing an effect for the first time, you will get a warning that says "Effekseer is from an unidentified developer...". 
Please refer to the following document and change the settings. You will be able to play the effect on the next run after setting it up.

https://www.macworld.co.uk/how-to/mac-software/mac-app-unidentified-developer-3669596/

エフェクトを初めて再生するときに「開発元を検証できないため開けません。」という警告が発生します。
下記の資料を参考に、設定を変更してください。設定した後の次の実行でエフェクトを再生できるようになります。

https://qiita.com/ganessa/items/0b6779e7b9b5b8e29a97
